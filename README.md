# 📌 Project Overview

The IPL Analytics Dashboard is an end-to-end Business Intelligence project developed using Microsoft Power BI.

The dashboard provides interactive insights into IPL matches, player performance, team statistics, venue analysis, and seasonal trends through dynamic visualizations and KPI reporting.

This project demonstrates skills in:

- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Calculations
- Interactive Dashboard Design
- Business Intelligence
- Data Storytelling

---

# 🎯 Business Objective

The objective of this project is to help cricket analysts, sports management teams, and fans understand:

- Team performance over multiple seasons
- Batting and bowling statistics
- Venue-wise match trends
- Toss decision impact
- Season-wise scoring patterns
- Top-performing players
- Team winning trends

# 📈 Business Insights

- Mumbai Indians have recorded the highest number of victories.
- Virat Kohli is among the highest run scorers in IPL history.
- Ravichandran Ashwin ranks among the leading wicket-takers.
- Winning the toss does not always guarantee winning the match.
- Certain venues consistently host more IPL matches than others.
- Team scoring trends have increased over recent seasons.
